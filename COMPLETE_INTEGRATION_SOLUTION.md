## 🌐 How Businesses Can Integrate

### Basic Integration (30 Seconds)
```html
<script src="https://turbo-answer-ai.uc.r.appspot.com/widget/turbo-widget.js"></script>
<script>
TurboWidget.init({
    primaryColor: '#3b82f6',
    position: 'bottom-right'
});
</script>
```

### E-commerce Store
```html
<script>
TurboWidget.init({
    primaryColor: '#10b981',
    welcomeMessage: 'Hi! Need help finding products?',
    position: 'bottom-right'
});
</script>
```

### SaaS Platform
```html
<script>
TurboWidget.init({
    primaryColor: '#8b5cf6',
    welcomeMessage: 'Welcome! How can I help you get started?',
    size: 'large'
});
</script>
```

### WordPress Integration
```php
<!-- Add to footer.php -->
<script src="https://turbo-answer-ai.uc.r.appspot.com/widget/turbo-widget.js"></script>
<script>
TurboWidget.init({
    primaryColor: '<?php echo get_theme_mod('primary_color', '#3b82f6'); ?>',
    welcomeMessage: 'Hi! How can I help you today?'
});
</script>
```
