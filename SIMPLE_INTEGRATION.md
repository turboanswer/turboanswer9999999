# Add AI to Your Website in 1 Step

## Copy This Code

Paste this before `</body>` on your website:

```html
<script src="https://turbo-answer-ai.uc.r.appspot.com/widget/turbo-widget.js"></script>
<script>TurboWidget.init();</script>
```

That's it! Your AI chat is now live.

## Change the Color

Want to match your brand? Change the color:

```html
<script>
TurboWidget.init({
    primaryColor: '#ff0000'  // Your brand color
});
</script>
```

## Examples

**Blue:** `primaryColor: '#3b82f6'`  
**Green:** `primaryColor: '#10b981'`  
**Purple:** `primaryColor: '#8b5cf6'`  
**Red:** `primaryColor: '#ef4444'`
