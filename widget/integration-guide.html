🚀 Turbo Answer AI Widget

Embed powerful AI assistance into any website in minutes

🎯 Quick Integration (2 Minutes)

Add this single line of code to your website:

<script src="https://turbo-answer-ai.uc.r.appspot.com/widget/turbo-widget.js"></script> <script> TurboWidget.init({ apiKey: 'your-api-key-here', primaryColor: '#3b82f6', position: 'bottom-right' }); </script>

That's it! Your AI assistant is now live on your website.

🎨 Live Demo

Click the blue chat button in the bottom-right corner to see the widget in action!

⚙️ Configuration Options

Option Type Default Description
apiKey string null Your Turbo Answer API key for authenticated access
primaryColor string #3b82f6 Widget color to match your brand
position string bottom-right Widget position: bottom-right, bottom-left, top-right, top-left
theme string auto Color theme: light, dark, auto
size string medium Widget size: small, medium, large
